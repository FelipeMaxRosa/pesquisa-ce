This is an application for Windows CE for a supermarket chain to carry out a product price search in competing supermarkets and later generate a CSV file so that the ERP ARIUS system could read and process it internally and compare the results. prices. The programming language used was C#.

Esta é uma aplicação para Windows CE para uma rede de supermercados para que fosse feita uma pesquisa de preços de produtos em supermercados concorrentes e posteriormente fosse gerado um arquivo CSV para que o sistema ERP ARIUS pudesse ler e processá-lo internamente e fizesse a comparação dos preços. A linguagem de programação utilizada foi o C#.
